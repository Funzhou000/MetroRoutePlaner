## 📋 详细使用说明

### Web版本（推荐）

1. **启动服务器**
   ```bash
   python3 Web.py
   ```
   
2. **打开浏览器**
   在地址栏输入：`http://localhost:8888`
   
3. **查询路线**
   - 输入起点ID（支持自动补全）
   - 输入终点ID（支持自动补全）
   - 点击"查询最短路线"
   - 查看详细结果

4. **停止服务**
   在终端按 `Ctrl+C`

### CLI版本

1. **启动程序**
   ```bash
   python3 CLI.py
   ```

2. **选择操作**
   ```
   1. 查询路线
   2. 搜索站点
   3. 查看历史
   4. 导出结果
   5. 显示帮助
   0. 退出
   ```

3. **查询示例**
   ```
   请输入选项: 1
   输入起点站点ID: 127S
   输入终点站点ID: 137S
   ```

## ❓ 常见问题

**Q: GUI启动失败怎么办？**
A: 这在某些macOS版本上很常见。使用Web版本（`python3 Web.py`）或CLI版本（`python3 CLI.py`）。

**Q: Web版本如何关闭？**
A: 在启动Web.py的终端中按 `Ctrl+C`。

**Q: Web版本需要安装什么？**
A: 不需要安装任何额外的库，只需Python 3.6+。

**Q: 可以在手机上访问Web版本吗？**
A: 可以，但需要在局域网内使用 `python3 Web.py` 时改为监听所有网卡（需要修改代码中的 localhost）。
