## 📋 详细使用指南

### Web版本（推荐首选）

**启动步骤：**
1. 打开终端，进入项目目录
2. 运行 `python3 Web.py`
3. 看到 "✓ 请在浏览器中打开: http://localhost:8888"
4. 打开浏览器，访问 `http://localhost:8888`

**界面说明：**
```
┌─────────────────────────────────┐
│  🚇 NYC Metro Route Planner      │
│  纽约地铁线路规划器             │
├─────────────────────────────────┤
│ 起点ID: [输入框 ▼] (自动补全)   │
│ 终点ID: [输入框 ▼] (自动补全)   │
│ [查询最短路线按钮]              │
├─────────────────────────────────┤
│ 总耗时: XX分XX秒  站点数: XX     │
│ 完整路线:                        │
│ 1. 127S - Rector St             │
│ 2. 128S - Cortlandt St          │
│ ...                             │
└─────────────────────────────────┘
```

**使用示例：**
1. 在"起点ID"输入框输入 `127` → 自动显示匹配的站点
2. 选择 `127S - Rector St, 1 Line`
3. 在"终点ID"输入框输入 `137` → 自动显示匹配的站点
4. 选择 `137S - 86th St, 4/5/6 Lines`
5. 点击"查询最短路线"按钮
6. 查看结果（通常< 1秒）

**特点：**
- ✨ 自动补全功能（输入时显示建议）
- ✨ 实时搜索
- ✨ 美观的网页界面
- ✨ 响应式设计（支持手机）
- ✨ 完整的兼容性

### CLI版本（功能最完整）

**启动步骤：**
1. 运行 `python3 CLI.py`
2. 看到菜单
3. 选择操作（输入 1-5）

**菜单说明：**
```
选择操作:
  1. 查询路线           - 输入起点和终点，计算最短路线
  2. 搜索站点           - 搜索特定站点
  3. 查看历史           - 查看之前的所有查询
  4. 导出结果           - 将查询结果保存为文件
  5. 显示帮助           - 显示详细说明
  0. 退出               - 关闭程序
```

**使用示例（查询路线）：**

```
请输入选项 (0-5): 1

终点: 137S - 86th St, 4/5/6 Lines
总耗时: 850 秒 (约 14 分 10 秒)
站点数: 15
======================================================================

完整路线:

  1. 127S      - Rector St, 1 Line
  2. 128S      - Cortlandt St
  3. 129S      - Park Place
  ...
```

**使用示例（搜索站点）：**

**特点：**
- ✨ 完整的交互式菜单

## ❓ 常见问题

**Q: Web版本需要安装什么吗？**
A: 不需要，仅需 Python 3.6+，其他都是内置库。

**Q: Web版本在哪打开？**
A: 启动后，在浏览器中输入 `http://localhost:8888`

**Q: 如何关闭Web版本？**
A: 在启动的终端中按 `Ctrl+C`

**Q: CLI版本如何退出？**
A: 选择 "0. 退出" 或按 `Ctrl+C`

**Q: 三个版本的功能是否相同？**
A: 核心功能相同，CLI版本额外有历史记录和导出。

**Q: 可以同时运行多个版本吗？**
A: 可以，但Web版本需要修改端口号（如8889）。
